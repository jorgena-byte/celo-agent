📈 CELO Auto Trading
Monitors CELO price every 5 minutes and executes trades automatically from the Safe wallet.
BUY trigger CELO drops -5% from baseline
SELL trigger CELO rises +10% from baseline
Trade size $100 cUSD per trade
Sell portion 40% of CELO balance
Stop loss -15% from baseline
Weekly DCA $50 cUSD every Monday 8AM
Check interval Every 5 minutes
HOW IT WORKS
A baseline price is set when the bot starts. Every 5 minutes it checks the current CELO price. If it drops 5% it buys $100 worth of CELO using cUSD from the Safe wallet. If it rises 10% it sells 40% of the CELO balance back to cUSD. If it drops 15% the stop loss triggers and the baseline resets.
💱 FX Yield Strategy
Exploits price differences between Celo stablecoins and real-world FX rates to generate yield.
Pairs monitored cEUR/cUSD, cREAL/cUSD
Min spread 0.5% vs real FX rate
Trade size $100 per opportunity
Cooldown 30 min per pair
FX data source open.er-api.com (live)
Price source GeckoTerminal (on-chain)
Execution Uniswap V3 via Safe wallet
HOW IT WORKS
Every 5 minutes the bot compares the on-chain exchange rate between cEUR/cUSD and cREAL/cUSD against real world FX rates. When the on-chain rate deviates more than 0.5% from the real rate, it executes a swap to capture the difference. A 30-minute cooldown prevents over-trading the same pair.
